contains_xray_crystallography_cell_data_solvationbox_padding_nm

default: 1.0
type: float
title: Contains xray crystallography cell data solvationbox padding nm.
short description: If the xray crystallography cell data size is available, add an additional solvationbox padding in nm.
long description: If the xray crystallography cell data size is available, add an additional solvationbox padding in nm. If the molecules contained inside xray crystallography cell are close to the cell edge, with periodic boundary conditions the atoms of the one side of the molecule will interact with the other side of itself.

explevel: easy

move_atoms_to_solvationbox_center

default: False
type: boolean
title: Move atoms to solvationbox center.
short description: Move atoms to solvationbox center, the pdb is placed around the origin, and water molecules of the solvation box are build up from the origin.
long description: Move atoms to solvationbox center, the pdb is placed around the origin, and water molecules of the solvation box are build up from the origin. Not centering the atoms shouldn’t be a problem if periodic boundary conditions are deployed, but employing it also prevents the molecule from jumping around the boundary while visually inspecting the simulation in e.g. PyMOL.
group:
explevel: easy
