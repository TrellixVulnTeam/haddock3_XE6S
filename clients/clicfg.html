Retrieve config

Extract the default configuration file for each module.

By default presents the parameters from the three expertise levels:

basic, intermediate, and guru.

Usage:

haddock3-cfg  # prints only the general parametes
haddock3-cfg -g  # same as above
haddock3-cfg -m MODULE  # prints only the module parameters
haddock3-cfg -m MODULE -l LEVEL
haddock3-cfg -m topoaa -l all
haddock3-cfg -m rigidbody -l all -g  # prints the module and the globals


usage: haddock3-cfg [-h]
                    [-m {caprieval,clustfcc,clustrmsd,emref,emscoring,exit,flexref,gdock,lightdock,mdref,mdscoring,rigidbody,rmsdmatrix,seletop,seletopclusts,topoaa,topocg}]
                    [-l {easy,expert,guru,all}] [-g]

Named Arguments

-m

Possible choices: caprieval, clustfcc, clustrmsd, emref, emscoring, exit, flexref, gdock, lightdock, mdref, mdscoring, rigidbody, rmsdmatrix, seletop, seletopclusts, topoaa, topocg

The module for which you want to retrieve the default configuration.

-l

Possible choices: easy, expert, guru, all

The expertise level of the parameters. Defaults to “all”.

Default: “all”

-g, --globals

Add also the optional module’s general parameters.

Default: False
