Parameters

  • segid (int, optional) – The ID of the segment. Defaults to None.

  • no_parent (boolean) – Whether to add the parent path .. to the :py:attribute:`haddock.libs.libstructure.Molecule.with_parent`. When set to true, the with_parent attribute returns the same as file_name.
